# OnePieceCanal

Sube `index.html` a la raíz de un repositorio y activa GitHub Pages.

## Importante
Para que funcione de verdad con Firebase:
1. En Firebase activa Firestore.
2. Crea la colección `videos` al guardar la primera entrada o deja que se cree sola.
3. Tus reglas de Firestore deben permitir leer y escribir desde tu web.

Si todavía no tienes reglas, la página puede cargar pero no guardar.
